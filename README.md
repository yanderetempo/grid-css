
Учебный проект по верстке 5 различных сеток (Bento Grid) по макету.

### Особенности:
- **Технологии:** HTML и CSS Grid.
- **Порядок:** Макеты расположены на странице снизу вверх (через `flex-direction: column-reverse`).
- **Стиль:** Острые углы, контрастные рамки и яркие цвета для наглядности структуры.

### Как запустить:
Открыть файл `index.html` в любом браузере.
